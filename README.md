# 🚀 エンゲージ求人自動抽出・通知システム

## 📌 概要
このアプリは、求人サイト「エンゲージ」から特定の条件に合致するエンジニア求人を自動で抽出し、
重要な案件のみを LINE Notify 経由でプッシュ通知するツールです。

## 🛠 使用技術
- **Python 3.x**
- **Selenium**: ブラウザ自動操作・スクレイピング
- **LINE Notify API**: 抽出結果のスマホ通知
- **GitHub Actions**: (今後実装予定) 定期実行の自動化

## ✨ こだわりポイント
- **独自スコアリング**: 「Java」「那覇」「未経験」などのキーワードを重み付けし、優先度の高い順に並び替え。
- **NGワードフィルタ**: IT事務やコールセンター、飲食業などのノイズを徹底排除。
- **ログイン維持**: ブラウザプロファイルを固定し、二回目以降のログイン入力を省略。

## 📦 セットアップ
1. `pip install selenium webdriver-manager requests`
2. LINE Notify のトークンを取得し、コード内に貼り付け
3. `python "job hunt app.py"` を実行
